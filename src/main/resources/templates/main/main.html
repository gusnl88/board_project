SpringBoot+Vue.js 학습용 CRUD


스프링 과 Vue.js 를 이용한 사용자 CRUD와 세션저장과 암호화

Vue.js는 사용자 인터페이스(UI)를 만들기 위한 강력한 도구입니다. 저는 Vue.js를 사용하여 사용자 목록을 표시하고, 새로운 사용자를 추가하며, 사용자 정보를 편집하고 삭제할 수 있는 화면을 만들었습니다. 각각의 작업을 Vue 컴포넌트로 나누어 구현하였고, 데이터는 사용자 경험을 향상시키기 위해 동적으로 업데이트되었습니다.

데이터 통신: Axios의 역할

서버와의 통신은 모던 웹 애플리케이션의 핵심입니다. Axios를 사용하여 HTTP 요청을 수행하면 서버와 데이터를 주고받을 수 있습니다. 이를 통해 사용자 데이터를 백엔드 서버와 동기화하고, 비동기 요청을 효과적으로 처리했습니다. 데이터는 애플리케이션과 서버 간에 원활하게 오고갔고, 사용자는 실시간으로 업데이트된 데이터를 볼 수 있었습니다.

상태 관리: Vuex의 힘

Vuex는 상태 관리 라이브러리로, 상태를 중앙에서 효과적으로 관리할 수 있게 도와줍니다. $store.commit을 사용하여 상태를 변경하고, 여러 컴포넌트 간에 데이터를 공유했습니다. 이를 통해 데이터의 일관성을 유지하고, 복잡한 애플리케이션에서도 데이터 관리를 단순하게 유지할 수 있었습니다.

데이터 보안: 암호화의 필요성

데이터의 보안은 항상 중요한 문제입니다. 사용자 데이터를 더 안전하게 저장하려고 window.btoa를 사용하여 데이터를 Base64로 인코딩했습니다. 이렇게 하면 데이터가 전송 중에 노출되더라도 더 안전하게 보호됩니다. 또한, 서버와의 통신에서 HTTPS를 사용하여 데이터를 암호화하여 더 높은 수준의 보안을 제공했습니다.
